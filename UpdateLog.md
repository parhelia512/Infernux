# Infernux 0.2.0

## Highlights
- Added a Godot 1.0-style animation controller foundation with 2D, 3D, and Timeline FSM workflows.
- Added `.animtimeline` assets, a Timeline editor, Timeline FSM runtime, and the `TimelineAction` component.
- Added blend-node support, animation events, expanded FSM editor tooling, and stronger animation runtime tests.
- Improved material, mesh, model, prefab, and Timeline preview reliability and performance.
- Added native fast-path transform updates for Timeline playback to reduce Python-side runtime overhead.
- Improved Project panel navigation boundaries, editor theming, asset preview styling, and inspector workflows.

## Packaging
- Bumped package metadata to `0.2.0`.
- Added automated GitHub Release publishing for version bumps.
- Release assets include the Windows wheel and `InfernuxHubInstaller.exe`.

Infernux 0.2.9 · Windows

Get started

Install InfernuxHub, choose an engine version, create a project, and run a scene. That is enough for a first pass.

1

Install InfernuxHub

Download and open InfernuxHub. The Hub manages engine versions, projects, and the Python runtime used by each project.

Download InfernuxHub
2

Install an engine version

  1. Open Installs in InfernuxHub.
  2. Choose the Infernux version you want and install it.
  3. Open Projects, create a project, and select that engine version.

You can keep more than one engine version installed and choose a version per project.

3

Know the editor windows

Hierarchy
Objects in the current scene.
Scene
Move, rotate, and arrange objects.
Game
What the active Camera renders.
Inspector
Components and values on the selected object.
Project
Scenes, scripts, materials, textures, and other assets.
Console
Errors, warnings, and your script logs.
4

Run a scene

  1. Create or open a scene.
  2. Add a Cube, a Camera, and a light if the scene is empty.
  3. Save the scene with Ctrl + S.
  4. Press Play and check the Game window.

If the Game window is empty, first check that the scene has an enabled Camera and then read the first error in Console.

5

Add a Python component

Create Spin.py in your project, then add Spin to a GameObject from the Inspector.

from Infernux import InxComponent, Vector3, serialized_field


class Spin(InxComponent):
    speed: float = serialized_field(default=90.0, range=(0.0, 360.0))

    def update(self, delta_time: float) -> None:
        self.transform.rotate(Vector3(0.0, self.speed * delta_time, 0.0))

Save the scene and enter Play mode. The object should rotate, and Speed should be editable in the Inspector.

开始使用

安装 InfernuxHub，选择一个引擎版本，创建项目，然后运行场景。第一次使用先做到这些就够了。

1

安装 InfernuxHub

下载并打开 InfernuxHub。Hub 用来管理引擎版本、项目，以及每个项目使用的 Python 运行环境。

下载 InfernuxHub
2

安装引擎版本

  1. 在 InfernuxHub 里打开安装页面。
  2. 选择需要的 Infernux 版本并安装。
  3. 回到项目页面，创建项目，并为项目选择刚才安装的版本。

你可以同时保留多个引擎版本，每个项目也可以选择不同的版本。

3

认识编辑器窗口

Hierarchy
当前场景里的物体。
Scene
移动、旋转和摆放物体。
Game
当前 Camera 实际渲染的画面。
Inspector
查看和修改选中物体的组件与数值。
Project
场景、脚本、材质、贴图等项目文件。
Console
错误、警告和脚本日志。
4

运行一个场景

  1. 创建或打开一个场景。
  2. 如果场景是空的，添加 Cube、Camera 和灯光。
  3. 按 Ctrl + S 保存场景。
  4. 点击 Play，然后查看 Game 窗口。

如果 Game 窗口没有画面，先确认场景里有启用的 Camera，再查看 Console 里的第一条错误。

5

添加 Python 组件

在项目里创建 Spin.py，然后从 Inspector 把 Spin 添加到一个 GameObject 上。

from Infernux import InxComponent, Vector3, serialized_field


class Spin(InxComponent):
    speed: float = serialized_field(default=90.0, range=(0.0, 360.0))

    def update(self, delta_time: float) -> None:
        self.transform.rotate(Vector3(0.0, self.speed * delta_time, 0.0))

保存场景并进入 Play。物体应该开始旋转，Inspector 里也应该能直接修改 Speed。

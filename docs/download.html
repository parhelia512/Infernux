Windows 10/11 · x64

下载 Infernux

先安装 InfernuxHub。项目和已经安装的引擎版本都可以在 Hub 里统一管理。

推荐

InfernuxHub

通过 Hub 安装或切换 Infernux 版本、创建项目、打开已有项目。普通用户不需要自己准备 Python 环境。

下载 InfernuxHub Windows x64 · InfernuxHub 0.2.9

选择引擎版本

InfernuxHub 可以同时安装多个引擎版本。需要直接下载 wheel 或使用旧版本时，在下面选择版本。

打开这个版本的下载页面

直接下载的引擎包面向 Windows x64 与 CPython 3.12。普通用户直接在 InfernuxHub 里安装版本即可。

安装之后

打开 InfernuxHub，在安装页面选择引擎版本，然后使用这个版本创建项目。

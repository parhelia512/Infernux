Before ignition

Know the preview boundary.

System

Windows 10 or 11, 64-bit, with a current Vulkan-capable graphics driver. The standalone wheel requires CPython 3.12 x64.

Preview status

APIs and project formats are still evolving. Keep projects in version control and read the Roadmap before production commitments.

Recovery

If verification or installation fails, retain the filename, checksum, Windows version, GPU driver, and exact error before opening an Issue.

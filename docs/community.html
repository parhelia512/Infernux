Data & privacy

Know what leaves this page.

Infernux has no forum account or post database. The page uses a few named external services, and GitHub remains the public record.

LOCAL-01

Local preferences

This site stores language, theme, optional learning-path progress, and up to eight recent document links in localStorage. Recent links expire after 180 days. Public topic metadata is cached for five minutes, and choosing “Load replies” is remembered only in the current tab via sessionStorage. It sets no first-party account cookie and keeps no user profile.

PUBLIC-02

Public GitHub content

The topic relay reads public Discussion metadata from GitHub without a token. Posts, replies, Issues, authors, and timestamps remain public and are stored on GitHub.

AUTH-03

Sign-in boundary

Giscus uses GitHub OAuth for comments and may keep a server-encrypted sign-in token in giscus.app localStorage. Infernux receives neither your password nor that access token.

NET-04

External requests

The public topic list contacts GitHub's API. giscus.app is contacted only after you choose “Load replies”; fonts and icons remain on the Infernux origin.

DATA CONTRACT

One identity, one moderation surface, no duplicate content store.

  • GitHub Discussions stores topics, answers, reactions, and the embedded wall.
  • GitHub Issues stores reproducible bugs and actionable engineering tasks.
  • GitHub permissions and moderation remain the authority; the website is a presentation layer.
  • ChenlizheMe is the sole declared Infernux community administrator; effective moderation authority is still enforced by GitHub repository permissions.

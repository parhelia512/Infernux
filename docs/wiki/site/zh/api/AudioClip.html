AudioClip¶

类位于 Infernux.core

描述¶

音频剪辑资源。

description 状态： Preview · 验证版本： 0.2.1

当前可靠解码器支持 WAV。当 AudioSource Track 或 One-shot 仍可能引用时，应保持 Clip 已加载。
